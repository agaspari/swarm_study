# Swarm Intelligence Study

A personal study project exploring swarm intelligence algorithms through interactive visualization.

## About

This project implements and visualizes various swarm intelligence optimization algorithms as documented in:

> **Swarm Intelligence: Principles, Advances, and Applications**  
> by Aboul Ella Hassanien & Eid Emary
